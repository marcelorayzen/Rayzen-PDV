# Guia do Garçom — Rayzen PDV

Como usar o sistema de atendimento pelo celular ou tablet.

---

## Antes de começar

Você vai usar o **navegador** do seu celular ou tablet (Chrome, Safari, qualquer um).

Peça ao gerente o **endereço** do sistema. Ele aparece na tela do caixa e tem esse formato:

```
http://192.168.1.100:3030
```

O número vai ser diferente em cada estabelecimento.

> O celular precisa estar no **Wi-Fi do estabelecimento** — o mesmo da máquina do caixa.

---

## Passo a passo do atendimento

### 1. Abrir o sistema

- Abra o navegador no celular.
- Digite o endereço (ex: `http://192.168.1.100:3030`) e pressione Enter.
- A tela de login vai aparecer.

---

### 2. Fazer login

- Digite o seu **PIN** (o mesmo que você usa no caixa).
- Toque em **Entrar**.

Você vai ver a tela principal de atendimento.

> O login dura **8 horas**. Depois desse prazo, ou se o sistema reiniciar, você precisará fazer login novamente.

---

### 3. Abrir uma comanda

Na tela principal:

1. Digite o **número da comanda** no campo "Comanda".
2. Se houver mesa, digite o **número da mesa** no campo "Mesa" (opcional).
3. Toque em **Buscar / Abrir**.

Se a comanda já existir, ela será carregada. Se não existir, uma nova será criada.

---

### 4. Adicionar itens

Depois de abrir a comanda:

1. Toque em uma **categoria** (ex: Bebidas, Lanches) para filtrar os produtos.
2. Toque no produto desejado para adicioná-lo.
3. O item aparece na lista de pedidos na parte de baixo da tela.

Repita para cada item que o cliente quiser.

---

### 5. Enviar para produção

Quando terminar de anotar todos os itens:

- Toque em **Enviar para produção**.

O pedido é enviado imediatamente para a cozinha ou bar, conforme o setor de cada produto.

Após enviar, a tela volta para o campo de comanda — pronto para atender o próximo cliente.

---

### 6. Atender o mesmo cliente depois

Se o cliente quiser pedir mais alguma coisa depois:

1. Digite o **mesmo número da comanda** de antes.
2. Toque em **Buscar / Abrir**.
3. Adicione os novos itens e envie novamente.

Os itens já enviados continuam na comanda normalmente.

---

## Dúvidas comuns

**O celular não abre o endereço.**
→ Verifique se o Wi-Fi do celular está conectado na rede do estabelecimento. Não funciona com dados móveis.

**Aparece "Sessão expirada".**
→ O login expirou. Volte para a tela de início e faça login novamente com seu PIN.

**O produto que quero não aparece.**
→ O produto pode não estar cadastrado ou estar inativo. Fale com o gerente.

**Enviei o pedido errado.**
→ O cancelamento de itens é feito pelo caixa na máquina principal. Avise o caixa ou gerente.

**A tela ficou em branco ou travou.**
→ Recarregue a página (puxe para baixo ou pressione F5). Se persistir, feche e abra o navegador novamente.

---

## Resumo rápido

```
Abrir navegador
  → digitar endereço do sistema
  → fazer login com PIN
  → digitar número da comanda → Buscar
  → selecionar categoria → tocar no produto
  → Enviar para produção
  → pronto para o próximo cliente
```
