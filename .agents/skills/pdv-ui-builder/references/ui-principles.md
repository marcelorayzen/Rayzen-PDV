# Principios de UI (PDV Rayzen)

- Teclado-first: foco inicial onde o operador precisa agir.
- Densidade: listas compactas, total e status sempre visiveis.
- Feedback operacional: offline, erro e impressora devem aparecer sem caca visual.
- Erros reversiveis: desfazer ou cancelar com motivo quando necessario.
